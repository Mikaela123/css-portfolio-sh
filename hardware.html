Hardware

Motherboard

The motherboard is a circuit board that contains all the other parts of the
computer. It is located in the inside of the computer. The motherboard is in
charge of holding the CPU, RAM, and all other computer hardware
components. Through this, each component is able to communicate with
one another because they are all connected and apart of the motherboard.

CPU

The Central Processing Unit, otherwise known as the CPU is considered
the brains of a computer. The CPU is the piece of hardware that carries
out the instructions of a computer program. This process begins when the
CPU receives information from the RAM and the CPU outputs the
commands on to the computer or hardware.

RAM

Where the CPU is considered the brain of the computer and the the RAM
is the memory. A random access memory (RAM) is a type of memory
storage that allows memory to be read through the computer. The RAM is a
place in a computer where current used data is kept so that the computer's
processor can quickly access it.

Hard Drive

The hard drive is the computer's memory storage. Although the hard drive
is responsible to hold your documents, pictures, music, and videos. They
hold your programs, files, preferences, and even your operating system.
The unfortunate part of your hard drive holding all your information, is that if
it crashes you can lose all your data.

How do they Commmunicate

All the components of the computer communicate with each other through
the motherboard. They send electrical signals to each other through the
wires that connect them to the motherboard. The RAM of the computer can
store current used data so that the computer processor can
access it easily. The CPU then receives the instructions from the RAM, and
sends back the results to output the commands.

Keyboard

The keyboard is an instrument with keys that consist of letters, numbers,
and punctuation which allows you to input your information into the
computer.

Monitor

The monitor of a computer is the display device that shows the graphical
information that was commanded. The monitor is connected to the
motherboard where all the components communicate with each other to
output the commands onto the screen of the monitor.

Mouse

The mouse is a usually small device used to help the user interact with the
computer. Through the use of a mouse you can move the device and the
monitor mirrors your movements onto the computer. The mouse is used to
select items on the monitor as well.
